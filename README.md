# Nordix Roleplay 2.0 – Server Files

Velkommen til **Nordix Roleplay 2.0** 👑
Dette repository indeholder de officielle serverfiler til Nordix Roleplay 2.0 – bygget til seriøs, realistisk og stabil FiveM RP.

---

## 📌 Om Projektet

Nordix Roleplay 2.0 er en videreudvikling af serverens tidligere version med:

* Optimeret performance
* Forbedret struktur
* Nye scripts og systemer
* Bedre RP-flow
* Stabil og skalerbar opsætning

Serveren er designet til kvalitets-RP med fokus på realisme, struktur og et stærkt community.

---

## ⚙️ Framework & Teknologi

* **Framework:** ESX
* **Database:** MySQL (oxmysql anbefales)
* **Voice:** pma-voice
* **Target System:** ox_target
* **Inventory:** ox_inventory (hvis inkluderet)

---

## 🚀 Installation

1. Installer en ren FiveM server build.
2. Upload alle filer til din server.
3. Importér SQL-filer til din database.
4. Konfigurer:

   * `server.cfg`
   * Database credentials
   * License key
5. Start serveren.

---

## 🔐 Krav

* Gyldig FiveM license key
* MySQL database
* Node.js (hvis scripts kræver det)
* Minimum 8GB RAM anbefales

---
